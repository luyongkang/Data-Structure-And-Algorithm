数据结构是由四个部分组成的：数据，逻辑结构，存储结构/物理结构，操作
1.逻辑结构就是指数据之间的逻辑关系：
	一共有四种：
	线性结构，树形结构，图形结构，集合结构（所有数据都是这四种结构的一种）
	其中树形结构和图形结构也被统称为非线性结构

2.存储/物理结构就是指数据在实际存储中的关系：
	一共就两种：
	顺序存储结构，链式存储结构

	每种逻辑结构都可以用这两种存储结构实现。



	线性结构：数据元素之间一对一（每个元素只有一个继承）
	树形结构：数据元素之间一对多（每个元素可以有多个继承）
	图形结构：数据元素之间多对多（可以有多个元素的继承是同一个元素，一个元素可以有多个继承）
	集合结构：数据元素之间不存在关系
